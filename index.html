Moy Yat Kung Fu Website

Moy Yat Kung Fu

Website Redesign

The main objective of this project was to redesign an outdated website for a local martial arts school. The school doubles as the headquarters for a global brand, where the lead instructor also serves as the Chair of the brand and organization. Key focuses included:

  • Brand and theme the website using the parent organization's existing branding scheme to provide consistency.
  • The site is completely responsive to all devices and screen sizes.
  • Create a simplistic and elegant interface that provides a sense of elegance. This was a common theme with the parent organization's founder, who was an artist and preferred direct and simple methods.
  • Provide a one page design for users to eliminate unneccesary content "clunkiness" that would distract from the user's experience.
  • Display historical content in a "presentation" style fashion that utilized the brand's colors.
Twitch.tv API Viewer

Twitch.tv Channel Viewer

Utilizing Twitch.tv's API

This project's objective was to use Twitch.tv's API and create a single page, channel viewer. The channels were pre-selected based on what channels were followed by one particular user.

  • The viewer shows if a channel is online or offline.
  • By default, all online channels are shown first, offline channels are secondary and unavailable channels are shown last.
  • A simple set of buttons on top allows fast and easy switching of views to see online and/or offline channels.
  • If a channel is online, a description of the content being streamed is provided.
  • Each channel presented is clickable by users and links directly to the actual channel on Twitch.tv.
  • The interace was based on TV programming guides that are popular among most people with cable or satellite TV.
  • The layout, even though simple, is responsive to various devices and screen sizes.
Wikipedia Search

Wikipedia Search

via Wikipedia's API

This project was to create a simplified search using Wikipedia's API. The interface was designed to emulate a typical search engine, with search results appearing in the same page.

  • Users are provided a simple interface to enter their search terms and receive results.
  • All results include a description of each returned result.
  • All search results are linked directly to an actual Wikipedia entry.
  • All results are filtered by most relevant to least relevant to the search terms entered.
  • Users are also offered a "Random Search" button on the interface to produce a random search entry.
  • All searches, results and descriptions are responsive to various devices and screen sizes.
Weather App

Weather App

Live Weather Based on Current Location

This web-based app is based on a user's current geolocation coordinates. Users are required to allow permission to check their geolocation via their browser and the app will automatically generate the weather information.

  • Coordinates are generated by the user's browser, when the user allows the webpage permission to get its current location.
  • All weather information is provided by an open weather API.
  • Users don't have to do anything beyond granting location access. The web app is self-working.
  • The temperature is available in both farenheit and celsius measurements. A simple blue button acts as a switch between the two.
  • The background wallpaper will change according to the temperature outside. Bright and warm wallpapers for warmer weather, cooler and darker photos for cooler weather.
  • The app is responsive to all devices and screen sizes.
Random Quote Machine

Random Quotes Generator

Random quotes selected from a list of various quotes.

This project was created as a simple application that generates (or displays) a quote selected from a pool of various quotes. This application does not rely on any APIs, but simply an internal list of pre-selected quotes built in.

  • The interface was designed to have just one button that provides the function of displaying a new quote each time it is clicked.
  • No API is used. A simple list of pre-selected quotes are built into the app.
  • Each quote also displays the quotee's name.
  • The app is responsive to all devices and screen sizes.
