About

Hey, there! My name is Andrew Kao.

Front End Developer, Developer Study Group Leader & Avid Learner.

I build websites for artists, musicians, bloggers and more. My passion for web development started back in 2001 after taking a class in web design. My first projects involved building websites and setting up MySpace pages for local bands.

Recently, I've redesigned a martial arts school website, created a custom WordPress theme for a NYC artist and built my own version of a weather app that tells you what to wear.

I absolutely love learning about technology. I enjoy volunteering each week as a study group leader for developers in the Bronx. We get together to share projects, debug code and pair program.

My front end web technologies:

  • HTML 5
  • CSS 3
  • Responsive Design / Bootstrap
  • JavaScript
  • jQuery
  • React
  • WordPress Theme Development

When I'm not coding and building pretty things, you'll find me geeking out over sci-fi movies and playing the guitar.
